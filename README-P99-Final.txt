Files included:
- p99_inventory_app_final.py : final Python app
- Run-P99-Inventory-App-Final.bat : runs the Python app directly
- build_console_exe.bat : builds a console EXE for debugging
- build_windowed_exe.bat : builds a normal windowed EXE

Recommended build order:
1. Run build_console_exe.bat
2. Test the EXE in the dist folder from PowerShell with:
   .\p99_inventory_app_final.exe
3. If it works, run build_windowed_exe.bat

Requirements on Windows:
- Python installed
- py launcher available
- Internet access for pip install, or packages already installed
